# 🌿 متجر  (خلاب) | Khalab  Shop

منصة إلكترونية لبيع النباتات، تركز على تجربة مستخدم عصرية ونظام إدارة مستخدمين متكامل.

##  مميزات العرض في الفيديو | Featured in Demo
* نظام المستخدمين: عرض حي لعملية إنشاء حساب جديد (Sign Up) وتسجيل الدخول (Login).
* التحقق من البيانات: استعراض قاعدة البيانات (MySQL) وكيفية تخزين بيانات المستخدمين فورياً.
* واجهة المستخدم (UI): جولة في الموقع تبرز الألوان المبهجة، الخلفيات الجمالية، وسهولة التنقل عبر الـ Navbar.
* التصميم المتجاوب: استعراض طريقة عرض المنتجات وتنسيقها البصري.

##  التقنيات المستخدمة | Tech Stack
* Front-end: HTML5, CSS3
* Back-end: PHP
* Database: MySQL

##  تشغيل المشروع محلياً | Local Setup
1. تأكدي من تشغيل XAMPP.
2. القيام باستيراد ملف قاعدة البيانات (المرفق مع الملفات) في phpMyAdmin.
3. وضع المجلد في الـ htdocs.
4. فتح  الرابط: http://localhost/Plant/index.php (أو اسم مجلدك).
